# Glimz

A mobile-first automotive service booking platform.

## What it does

- Customers book car services (washing, detailing, maintenance)
- Schedule appointments with service providers
- Manage vehicles and addresses
- Pay via wallet, coupons, or Moyasar gateway
- Earn loyalty points and buy service packages

## Tech Stack

- **Backend:** Laravel 12 (PHP 8.2+)
- **Frontend:** Vue.js + Vite + Tailwind CSS
- **Auth:** Laravel Sanctum (API tokens)
- **Payments:** Moyasar integration
- **SMS:** Twilio
- **Database:** MySQL

## Key Modules

- Services & Categories
- Bookings & Scheduling
- Employee management (work schedules, zones)
- Wallets & Transactions
- Coupons & Packages
- Notifications
- Multi-language (Arabic/English)
